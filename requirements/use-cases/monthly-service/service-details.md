# Summary

- **Title:** Monthly Service Details
- **Description:** The system gives details related to a monthly service
- **Pre-condition:** Monthly Service must exists
- **Pos-condition:** Monthly Service's details are shown

# Flow

### Normal Flow

1. User initiates the monthly service details process.
2. System provides informations about this monthly service, such as:
    - ID
    - Name
    - Description
    - Category Related
    - Money amount
    - Is it active?
